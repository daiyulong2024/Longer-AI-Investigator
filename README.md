## 📖 简介 | Introduction

**Longer AI Investigator** 是一个专为探索 AI 群体行为而设计的社会实验模拟平台。

你是否好奇：
- *如果让 50 个 AI 扮演面临高考压力的“高中生”，面对作弊诱惑，它们会如何选择？*
- *如果让 100 个 AI 扮演“企业经理”，在裁员潮中，它们会选择保全团队还是服从上级？*
- *AI 的决策是绝对理性的吗？还是也会表现出从众、犹豫甚至情感波动？*

在这里，你可以扮演**上帝视角**，自定义角色性格、构建极端场景、提出道德困境，然后释放成百上千个 AI Agent 投身其中。通过实时监控它们的决策过程，生成详细的统计报告，我们可以一窥 AI “内心”的社会学逻辑。

## ✨ 核心功能 | Features

### 🎭 深度角色定制 (Persona Engine)
不仅仅是简单的提示词。你可以定义角色的**行为逻辑**（理性分析、情感驱动、从众心理、随机游走等）和**详细背景故事**。
- **内置 30+ 模板**：涵盖医生、警察、法官、艺术家、电竞选手等各行各业，一键套用。
- **自定义 DNA**：完全自由地创造你想象中的任何角色。

### ⚡ 高并发模拟 (Parallel Simulation)
拒绝漫长的等待！
- **自定义并发**：支持设置并行线程数，根据你的设备性能榨干算力。
- **大规模实验**：轻松模拟 100+ 甚至更多 Agent 的群体决策。

### 👁️ 实时监控大盘 (Live Monitor)
像黑客帝国一样监控数据流。
- **决策日志流**：实时滚动显示每个 Agent 的决策推理过程。
- **动态进度条**：直观掌握实验进度。
- **防误触机制**：运行中自动锁定，防止意外退出导致数据丢失。

### 📊 专业级统计报告 (Pro Analytics)
用数据说话。
- **可视化图表**：自动生成选项分布柱状图，直观对比决策倾向。
- **详细推理记录**：查看每一个“异类”的选择理由，支持按选项排序、筛选。
- **CSV 导出**：一键导出所有实验数据，方便进行更深度的二次分析（如 Excel、Python）。

### 🚀 纯前端架构 (Zero Backend)
- **极致轻量**：没有复杂的服务器部署，没有数据库依赖。
- **数据隐私**：所有实验数据均存储在你的浏览器本地 (LocalStorage)，安全无忧。
- **API 集成**：支持配置 OpenAI 兼容接口（如 GPT-3.5/4），让模拟更加真实智能（同时也支持本地 Mock 模式体验）。

## 🛠️ 快速开始 | Quick Start

### 方法一：在线体验 (推荐) 🚀
无需下载，直接访问我的部署站点：http://longer26.1.t1xz.com/

### 方法二：本地运行 (Local) 💻
如果你想查看源码或离线使用：

1.  **克隆或下载本项目**
    ```bash
    git clone https://github.com/daiyulong2024/Longer-AI-Investigator.git
    ```
2.  **直接打开**
    进入文件夹，双击 `index.html` 在浏览器中打开即可！
    
    *(推荐使用 Chrome, Edge, Firefox 等现代浏览器)*
